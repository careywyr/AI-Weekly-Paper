## MolmoAct2: Action Reasoning Models for Real-world Deployment

[MolmoAct2：面向真实部署的动作推理模型](https://arxiv.org/abs/2605.02881)

MolmoAct2 是一个面向机器人领域的开放动作推理模型，通过专用视觉-语言模型（VLM）骨干网络、新数据集、开放权重的动作分词器（Action Tokenizer）、面向连续动作预测的架构重设计以及自适应推理来降低延迟，在多个维度上超越前代系统。视觉-语言-动作（VLA）模型旨在为机器人提供统一的通用控制器，但现有系统在面向真实世界部署的关键指标上仍存在不足：前沿模型闭源、开放权重替代方案依赖昂贵硬件、推理增强策略因延迟过高难以实用、微调成功率仍低于可靠使用的阈值。MolmoAct2 在五个方面进行了改进：推出了 MolmoER——一个专为空间和具身推理设计的 VLM 骨干网络，基于 330 万样本语料库采用"先专精后综合"的训练策略；发布了三个涵盖低成本到中等成本平台的新数据集，其中包括 MolmoAct2-BimanualYAM——720 小时的遥操作双臂轨迹数据，构成迄今最大的开放双臂数据集，以及经质量筛选的 Franka（DROID）和 SO100/101 子集；提供了 OpenFAST——一个开放权重、开放数据的动作分词器，在五种机器人形态上的数百万条轨迹上训练完成；重新设计了架构，通过逐层 KV-cache 条件机制将流匹配连续动作专家嫁接到离散 token VLM 上；提出了 MolmoThink——一种自适应深度推理变体，仅对时间步之间发生变化的场景区域重新预测深度 token，以远低于先前方案的延迟保持几何推理能力。在涵盖 7 个仿真和真实世界基准的、迄今最广泛的开放 VLA 实证研究中，MolmoAct2 超越了包括 Pi-05 在内的强基线，而 MolmoER 在 13 个具身推理基准上超越了 GPT-5 和 Gemini Robotics ER-1.5。项目已发布模型权重、训练代码和完整训练数据。

## From Context to Skills: Can Language Models Learn from Context Skillfully?

[从上下文到技能：语言模型能否从上下文中巧妙地学习？](https://arxiv.org/abs/2604.27660)

本文提出了 Ctx2Skill，一个自我进化框架，通过多智能体自博弈循环——包含 Challenger（挑战者）、Reasoner（推理者）和 Judge（评判者）——自主发现、优化和选择上下文特定的技能，无需人工监督或外部反馈即可提升上下文学习能力。许多现实任务要求语言模型在超出其参数化知识的复杂上下文中进行推理，这需要上下文学习——即直接从给定上下文中学习相关知识。一种直觉方案是推理时技能增强：从上下文中提取规则和程序形成自然语言技能。然而，为上下文学习场景构建此类技能面临两个挑战：对冗长、技术密集的上下文进行人工技能标注成本高昂，以及自动技能构建缺乏外部反馈。Ctx2Skill 的核心是一个多智能体自博弈循环：Challenger 生成探测任务和评分标准，Reasoner 在不断进化的技能集引导下尝试解题，中立的 Judge 提供二元反馈。Challenger 和 Reasoner 都通过积累的技能进化——专门的 Proposer 和 Generator 智能体分析失败案例并合成针对性的技能更新。为防止对抗性坍塌——即日益极端的任务生成和过度专业化的技能积累，进一步引入了 Cross-time Replay 机制，识别在代表性案例上达到最佳平衡的技能集。最终的技能可插入任何语言模型以获得更好的上下文学习能力。在 CL-bench 的四个上下文学习任务上，Ctx2Skill 持续提升了各骨干模型的解题率。

## Stream-R1: Reliability-Perplexity Aware Reward Distillation for Streaming Video Generation

[Stream-R1：面向流式视频生成的可靠性-困惑度感知奖励蒸馏](https://arxiv.org/abs/2605.03849)

Stream-R1 通过基于可靠性和困惑度自适应加权监督信号来改进视频扩散模型蒸馏，在不增加额外计算开销的情况下提升了视觉质量、运动质量和文本对齐度。基于蒸馏的加速方法已成为使自回归流式视频扩散模型实用化的基础，分布匹配蒸馏（DMD）成为事实标准。然而现有方法在训练学生模型时无差别地匹配教师模型的输出，将每次推演、每帧和每个像素视为同等可靠的监督。本文认为这限制了蒸馏质量，因为它忽视了 DMD 监督中两个互补的方差维度：跨学生推演的 Inter-Reliability（监督可靠性不同）和跨空间区域与时间帧的 Intra-Perplexity（对质量改进的贡献不等）。Stream-R1 在推演级别和时空元素级别通过共享的奖励引导机制自适应地重新加权蒸馏目标：在 Inter-Reliability 层面，通过预训练视频奖励分数的指数缩放各推演的损失，使具有可靠监督的推演主导优化；在 Intra-Perplexity 层面，反向传播同一奖励模型提取逐像素梯度显著性，分解为空间和时间权重，将优化压力集中在收益最大的区域和帧上。自适应平衡机制防止任何单一质量维度在视觉质量、运动质量和文本对齐中占据主导。在标准流式视频生成基准上，Stream-R1 在三个维度上均一致超越蒸馏基线，无需架构修改或额外推理成本。

## RLDX-1 Technical Report

[RLDX-1 技术报告](https://arxiv.org/abs/2605.03269)

RLDX-1 是一个面向灵巧操作的通用机器人策略，通过多流动作 Transformer（Multi-Stream Action Transformer, MSAT）架构整合异构模态，在复杂真实世界任务中展现出优于现有视觉-语言-动作（VLA）模型的性能。虽然 VLA 模型通过预训练视觉-语言模型继承的多功能智能（广泛的场景理解和语言条件泛化）在通用人形机器人策略方面取得了显著进展，但在需要更广泛功能能力的复杂真实世界任务上仍然力不从心，例如运动感知、记忆感知决策和物理感知。RLDX-1 的 MSAT 架构通过模态特定流和跨模态联合自注意力统一了这些能力。在系统层面，RLDX-1 还包括为罕见操作场景合成训练数据、专为类人操作设计的学习流程以及面向实时部署的推理优化。在 ALLEX 人形机器人任务中，RLDX-1 达到 86.8% 的成功率，而 Pi-0.5 和 GR00T N1.6 仅约 40%，展示了其在多样化功能需求下控制高自由度人形机器人的能力。

## ARIS: Autonomous Research via Adversarial Multi-Agent Collaboration

[ARIS：基于对抗性多智能体协作的自主研究系统](https://arxiv.org/abs/2605.03042)

ARIS（Auto-Research-in-sleep）是一个开源研究框架，通过跨模型对抗性协作确保长期研究结果的可靠性，包含执行层、编排层和保障层三个架构层次。基于 LLM 的智能体系统的性能不仅取决于模型权重，还取决于围绕它们的外围框架——控制存储、检索和呈现给模型的信息。对于长周期研究工作流，核心失败模式不是可见的崩溃，而是看似合理但缺乏支撑的"成功"：长期运行的智能体可能产出证据支持不完整、错误报告或悄然继承自执行器框架的主张。ARIS 通过跨模型对抗性协作协调机器学习研究工作流：执行器模型推进工作进度，来自不同模型家族的评审者对中间产物进行批判并请求修订。执行层提供超过 65 个可复用的 Markdown 定义技能、通过 MCP 的模型集成、用于迭代复用先前发现的持久研究 Wiki 以及确定性图表生成。编排层协调五个端到端工作流，具有可调节的工作量设置和可配置的评审模型路由。保障层包括三阶段实验声明证据支撑检查流程、五轮科学编辑管道、数学证明检查和 PDF 渲染视觉检查。

## Stream-T1: Test-Time Scaling for Streaming Video Generation

[Stream-T1：面向流式视频生成的测试时扩展](https://arxiv.org/abs/2605.04461)

Stream-T1 通过流式合成与时序引导实现高效的测试时视频生成，在一致性和质量上超越了现有基于扩散的方法。虽然测试时扩展（Test-Time Scaling, TTS）为在不增加训练成本的情况下提升视频生成质量提供了有前景的方向，但当前基于扩散模型的测试时视频生成方法面临过高的候选探索成本且缺乏时序引导。本文提出将焦点转向流式视频生成——其分块级合成和少量去噪步骤天然适合 TTS，显著降低了计算开销并支持细粒度时序控制。Stream-T1 由三个组件构成：（1）Stream-Scaled Noise Propagation，利用历史已验证的高质量先前分块噪声主动优化当前生成分块的初始隐噪声，通过高斯先验引导当前生成；（2）Stream-Scaled Reward Pruning，综合评估生成候选，通过整合即时短期评估与滑动窗口长期评估，在局部空间美感和全局时间连贯性之间取得最优平衡；（3）Stream-Scaled Memory Sinking，根据奖励反馈将 KV-cache 中驱逐的上下文动态路由到不同的更新路径，确保先前生成的视觉信息有效锚定和引导后续视频流。在 5 秒和 30 秒综合视频基准上，Stream-T1 在时间一致性、运动平滑度和帧级视觉质量方面均展现出显著优势。

## Continuous Latent Diffusion Language Model

[连续隐空间扩散语言模型](https://arxiv.org/abs/2605.06548)

Cola DLM 提出了一种层次化隐空间扩散语言模型，通过文本到隐空间的映射、连续隐空间中的全局语义先验建模和条件解码，实现了具有灵活非自回归归纳偏好的高效文本生成。大语言模型在自回归范式下取得了巨大成功，但高质量文本生成并不必然绑定于固定的从左到右顺序。现有替代方案仍难以同时实现生成效率、可扩展的表示学习和有效的全局语义建模。Cola DLM 首先通过 Text VAE 学习稳定的文本到隐空间映射，然后通过块因果 DiT 在连续隐空间中建模全局语义先验，最后通过条件解码生成文本。从统一的马尔可夫路径视角看，其扩散过程执行隐先验传输而非 token 级别的观测恢复，将全局语义组织与局部文本实现分离。这一设计产生了更灵活的非自回归归纳偏好，支持连续空间中的语义压缩和先验拟合，并自然扩展到其他连续模态。在涵盖 8 个基准、严格匹配的约 2B 参数自回归和 LLaDA 基线以及约 2000 EFLOPs 的缩放曲线上的实验表明，层次化连续隐先验建模是严格 token 级语言建模的原则性替代方案。

## MiA-Signature: Approximating Global Activation for Long-Context Understanding

[MiA-Signature：用于长上下文理解的全局激活近似](https://arxiv.org/abs/2605.06416)

研究者提出了一种面向大语言模型全局激活模式的压缩表示方法——Mindscape Activation Signature（MiA-Signature），通过基于子模块化的高级概念选择覆盖激活的上下文空间，并可选择性通过工作记忆进行轻量迭代更新。认知科学研究指出，可报告的意识访问与分布式记忆系统上的全局点火（Global Ignition）相关，但这种激活仅部分可访问——个体无法直接访问或枚举所有被激活的内容。这一张力暗示了一种可能的机制：认知可能依赖于一种紧凑表示来近似全局激活对下游处理的影响。MiA-Signature 作为条件信号，近似完整激活状态的效果，同时保持计算可行性。将其集成到 RAG 和智能体系统中，在多个长上下文理解任务上均获得了一致的性能提升。
