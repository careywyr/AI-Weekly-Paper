## Gamma-World: Generative Multi-Agent World Modeling Beyond Two Players

[Gamma-World: 超越双玩家的生成式多智能体世界建模](https://arxiv.org/abs/2605.28816)

本文提出了一种生成式多智能体世界模型，通过采用单纯形旋转智能体编码 (Simplex Rotary Agent Encoding) 和稀疏枢纽注意力 (Sparse Hub Attention)，在交互式视频生成中实现了多个智能体之间可扩展且排列对称的交互。现有的交互式视频生成世界模型主要针对单智能体场景，即所有未来观测均由单一控制信号生成。然而，许多生成环境需要多智能体交互：多个玩家、机器人或具身智能体需在同一共享空间内并发行动。将世界模型推广至此类场景需要一种原则性的多智能体设计：智能体应保持独立可控、排列对称，并支持高效推理，同时保证跨时间步与视角的一致性。本文提出的生成式多智能体世界模型专为交互式模拟而设计。其中引入了单纯形旋转智能体编码，这是 3D 旋转位置编码 (3D RoPE) 的一种无需参数扩展的方法，它将每个智能体表示为旋转角度空间中正单纯形的一个顶点。该编码为每个智能体分配了不同的相位，同时使所有智能体排列等价，从而无需学习每个插槽的身份或固定的智能体顺序，即可实现可扩展的智能体身份识别。为避免智能体之间采用稠密的全连接注意力机制，我们进一步提出了稀疏枢纽注意力：一组可学习的枢纽令牌 (hub token) 负责调解跨智能体的令牌交互，将跨智能体注意力计算成本从关于智能体数量的二次方降低为线性。为了实现实时推演，我们将一个全上下文扩散教师模型蒸馏为一个因果学生模型，该学生模型通过键值缓存 (KV caching) 逐块顺序生成时间块，从而能够以 24 FPS 的速度生成对动作有响应的视频。在多玩家虚拟环境中的实验表明，与基于插槽和稠密注意力的基线方法相比，我们的模型在视频保真度、动作可控性及智能体间一致性方面均取得提升，并且无需额外训练即可从两个玩家泛化到四个玩家。

## SkillOpt: Executive Strategy for Self-Evolving Agent Skills

[SkillOpt: 自进化智能体技能的优化策略](https://arxiv.org/abs/2605.23904)

SkillOpt 提出了一种系统性的文本空间优化方法，用于训练智能体技能。它将技能作为外部状态进行训练，实现稳定的更新过程，并在部署阶段不增加推理开销，在多个基准测试和执行环境中取得了显著性能提升。当前，智能体技能通常采用手工设计、一次性生成或松散控制的自我修订方式演化，这些方法均未采用类似深度学习优化器的工作机制，也无法在反馈信号的引导下可靠地改进初始状态。我们认为，技能应当作为冻结智能体的外部状态进行训练，遵循与权重空间优化相同的基本原则，确保可复现性。据我们所知，SkillOpt 是首个针对智能体技能的系统性可控文本空间优化器：一个独立的优化器模型将带得分的 rollout 结果转化为对单个技能文档进行受限的添加、删除或替换编辑操作，仅当编辑结果能显著提升留出验证集分数时，该编辑才会被接受。通过引入文本学习率预算、被拒绝编辑缓冲区以及逐 epoch 的慢/元更新机制，SkillOpt 在保持技能训练稳定性的同时，在部署阶段实现了零推理调用开销。在六个基准测试、七个目标模型和三个执行框架（直接对话、Codex、Claude Code）上，SkillOpt 在所有 52 个评估单元（模型、基准、框架的组合）上表现最佳或与最优结果持平，并在每个评估单元上击败了所有竞争对手，包括人类设计、单次 LLM 生成、Trace2Skill、TextGrad、GEPA 和 EvoSkill 等方法。在 GPT-5.5 上，SkillOpt 将直接对话场景中的无技能基线准确率提升了 +23.5 个百分点，在 Codex 智能体循环中提升了 +24.8 个百分点，在 Claude Code 中提升了 +19.1 个百分点。迁移实验进一步表明，优化后的技能制品在跨模型规模迁移、在 Codex 与 Claude Code 执行环境之间迁移，以及迁移到邻近的数学基准测试（无需进一步优化）时，其有效性仍能得以保持。

## DVAO: Dynamic Variance-adaptive Advantage Optimization for Multi-reward Reinforcement Learning

[DVAO: 面向多奖励强化学习的动态方差自适应优势优化](https://arxiv.org/abs/2605.25604)

动态方差自适应优势优化 (DVAO) 通过根据经验奖励方差自适应调整各目标的权重，来解决多奖励强化学习中的训练不稳定问题。该方法能维持优势幅度有界，从而提升多目标性能。强化学习已成为将大语言模型对齐人类意图和任务需求的标准范式。尽管组相对策略优化 (Group Relative Policy Optimization) 为近端策略优化 (Proximal Policy Optimization) 提供了一种高效且免值函数的替代方案，但将其适配到现实世界的多奖励场景中仍面临挑战。标准的标量化做法，例如奖励组合和优势组合，存在明显缺陷：奖励组合经常生成平方幅度过大的优势值，导致训练不稳定；而优势组合依赖于静态超参数，且忽略了目标间的相关性。为克服这些局限，我们提出了 DVAO。该方法会在每个采样组内，基于各目标的经验奖励方差动态调整组合权重，从而有效提升学习信号较强目标的权重，同时抑制噪声目标的权重。我们从数学上证明 DVAO 能保持优势幅度有界，从而确保训练稳定，并引入了一种自适应的跨目标正则化机制。在数学推理和工具使用基准上使用 Qwen3 和 Qwen2.5 模型进行的广泛实验表明，DVAO 显著优于基线方法，实现了更优越的多目标帕累托前沿和稳健的训练稳定性。

## LocateAnything: Fast and High-Quality Vision-Language Grounding with Parallel Box Decoding

[LocateAnything: 基于并行框解码的快速高质量视觉语言定位](https://arxiv.org/abs/2605.27365)

并行框解码（Parallel Box Decoding）通过将几何元素作为原子单元进行解码，实现了高效且准确的统一视觉定位与检测，同时提升了吞吐量和定位质量。视觉语言模型（VLM）通常将视觉定位与检测建模为坐标令牌生成任务，将每个二维框序列化为多个一维令牌，而这些令牌在多数情况下是独立学习与解码的。这种逐令牌解码方式并不匹配框几何结构的耦合特性，且由于严格顺序化的生成过程，在实际推理中形成了瓶颈。我们提出了LocateAnything，这是一个基于并行框解码（PBD）的统一生成式定位与检测框架。通过将边界框和点等几何元素作为原子单元在单步中解码，LocateAnything保持了框内几何一致性，并释放了显著的并行能力。实验表明，并行框解码不仅提高了解码吞吐量，也增强了定位精度。我们还构建了一个可扩展的数据引擎，并精心制作了LocateAnything-Data，该大规模数据集包含超过1.38亿个训练样本，显著增强了高精度定位任务的数据多样性。广泛的评估结果显示，LocateAnything在速度与精度方面取得了新的突破，在多个基准测试中，解码吞吐量显著提升的同时，高交并比（IoU）下的定位质量也得到了改善。这些结果突显了并行框解码与大规模训练数据在实现高效且精确的统一视觉定位与检测中的互补优势。

## AgentDoG 1.5: A Lightweight and Scalable Alignment Framework for AI Agent Safety and Security

[AgentDoG 1.5：一种轻量级且可扩展的AI智能体安全对齐框架](https://arxiv.org/abs/2605.29801)

## Rethinking Cross-Layer Information Routing in Diffusion Transformers

[重新思考扩散 Transformer 中的跨层信息路由](https://arxiv.org/abs/2605.20708)

扩散 Transformer (Diffusion Transformers) 面临层间信息流动低效的问题，而传统的残差连接无法有效解决。为此，我们引入了一种可学习、且能自适应时间步的路由机制，从而显著提升了训练效率与模型质量。扩散 Transformer (DiTs) 已成为现代视觉生成领域事实上标准的基础架构，其设计的几乎所有关键方面——包括 token 化、注意力机制、条件控制、损失函数以及潜空间自编码器——都已得到了广泛的重新审视。然而，负责控制信息在各层间累积方式的残差流，却直接沿用了原始 Transformer 的设计。在本文中，我们对 DiTs 中的跨层信息流进行了系统性的实证分析，同时从网络深度与去噪时间步两个维度展开，并识别出传统残差加法存在的三个具体问题：前向幅度单调膨胀、反向梯度急剧衰减以及明显的逐模块冗余。基于这一诊断，我们提出了扩散自适应路由 (Diffusion-Adaptive Routing, DAR)，这是一种可直接替换残差连接的新方案。它通过对各子层输出的历史信息进行可学习、时间步自适应且非递增的聚合操作。此外，所提出的 DAR 与许多现代的 Transformer 增强方法（如 REPA）兼容。在 ImageNet 256x256 数据集上，DAR 使 SiT-XL/2 的 FID 降低了 2.11（从 9.67 降至 7.56），并且仅需基线模型 8.75 分之一的训练迭代次数即可达到相同的收敛质量。当与 REPA 结合使用时，DAR 在训练早期阶段实现了 2 倍的加速效果，这揭示了跨层信息路由是扩散建模中一个此前未被充分探索的设计维度，其作用与现有的表示对齐目标正交。除了预训练阶段，DAR 也可应用于大规模文生图 (T2I) 模型的微调阶段，并在分布匹配蒸馏 (Distribution Matching Distillation) 过程中有效保留高频细节。

## Lens: Rethinking Training Efficiency for Foundational Text-to-Image Models

[Lens: 重新思考基础文生图模型的训练效率](https://arxiv.org/abs/2605.21573)

Lens 是一个紧凑的 3.8B 参数文本到图像 (Text-to-Image, T2I) 模型，通过密集标注数据集、多分辨率批处理 (Multi-Resolution Batching)、高效架构和优化技术，在显著降低训练计算量的同时实现了优越性能。我们提出的 Lens 模型仅 3.8B 参数，在各种基准测试中达到了与超过 6B 参数的最先进模型相竞争甚至在某些方面超越的性能，且所需训练计算量大幅减少。例如，Lens 仅需 Z-Image 所用训练计算量的约 19.3%。Lens 的训练效率除其紧凑模型尺寸外，还源于两大关键策略。第一，我们最大化每个训练批次的数据信息密度：(i) 在 Lens-800M 数据集上训练，该数据集包含 8 亿个密集标注的图像-文本对，所有标注由 GPT-4.1 生成，平均约 109 个单词，比传统短标注提供更丰富的语义监督；(ii) 从具有多种分辨率和不同宽高比的图像中构建每个批次，从而扩大每个优化步骤的有效视觉覆盖范围。第二，我们通过精心的架构选择提升收敛速度，包括采用能提供更好潜在表示的语义变分自编码器 (Semantic VAE)，以及使用强大的语言编码器，该编码器能加速优化过程，并支持从仅英语训练数据实现多语言泛化。预训练后，我们应用基于分类法提示的强化学习 (Reinforcement Learning, RL)（即 Lens-RL-8K）和结构化奖励评分标准来抑制伪影并提升视觉质量；还引入了一个推理模块 (Reasoner Module)，通过无需训练的系统提示搜索 (Training-Free System Prompt Search) 更好地将用户请求与模型对齐；以及基于蒸馏的加速方法用于 4 步推理。凭借高效训练和系统化优化，Lens 能够泛化到从 1:2 到 2:1 的任意宽高比和高达 1440^2 的分辨率，并支持多种常用语言的提示输入。得益其紧凑的尺寸，Lens 在单张 NVIDIA H100 GPU 上生成 1024^2 像素的图像仅需 3.15 秒，而其蒸馏快速版 (Distilled Turbo Version) 可在 0.84 秒内完成 4 步生成。

## Qwen-VLA: Unifying Vision-Language-Action Modeling across Tasks, Environments, and Robot Embodiments

[Qwen-VLA: 统一不同任务、环境和机器人本体的视觉-语言-动作建模](https://arxiv.org/abs/2605.30280)

本文提出了一个统一的视觉-语言-动作模型，通过共享架构和训练方法整合了多样化的具身决策任务，在操作、导航和轨迹预测任务中均展现出强劲的性能，并能泛化到不同的机器人平台和环境。当前，具身智能通常采用为操作或导航等特定任务分别设计的专用模型进行研究，这导致能力碎片化，且跨任务、环境和机器人本体的泛化能力有限。本文研究了一个核心问题：异构的具身决策问题能否被统一到单个视觉-语言-动作模型中。为此，我们提出了 Qwen-VLA，一个统一的具身基础模型。该模型将 Qwen 的视觉-语言建模栈从感知、理解和推理扩展到连续动作和轨迹生成，具体采用基于 DiT (Diffusion Transformer) 的动作解码器实现。Qwen-VLA 通过大规模联合预训练方案在多样化的数据源上进行训练，涵盖机器人操作轨迹、人类第一人称演示、合成仿真数据、视觉-语言导航数据、轨迹任务相关的监督数据以及辅助的视觉-语言数据。为了支持多种机器人平台，我们引入了具身感知提示条件化 (embodiment-aware prompt conditioning) 机制，利用特定于机器人的文本描述来指定当前的具身形态和控制惯例。我们进一步将操作、导航和轨迹预测统一到一个动作和轨迹联合预测框架中，从而实现跨机器人形态、任务族和环境的可迁移视觉定位、空间推理和连续动作生成能力。在操作、导航和以轨迹为中心的基准测试上的实验表明，当场景布局、背景、光照、物体配置和机器人具身形态发生变化时，模型依然保持了稳定的多任务性能和分布外 (OOD) 泛化能力。具体而言，Qwen-VLA-Instruct 模型在 LIBERO 基准上达到 97.9%，在 Simpler-WidowX 上达到 73.7%，在 RoboTwin-Easy/Hard 上分别达到 86.1%/87.2%，在 R2R 视觉导航任务上获得 69.0% 的目标到达率 (OSR)，在 RxR 任务上获得 59.6% 的任务成功率 (SR)，在真实世界 ALOHA 操作实验中平均分布外 (OOD) 成功率为 76.9%，在 DOMINO 动态操作任务上实现 26.6% 的零样本成功率。

## WBench: A Comprehensive Multi-turn Benchmark for Interactive Video World Model Evaluation

[WBench: 面向交互式视频世界模型评估的综合多轮基准](https://arxiv.org/abs/2605.25874)

WBench 提出了一套全面的多轮基准，用于在五个维度上评估交互式世界模型，该基准包含 289 个测试用例和 1,058 个交互轮次，覆盖多样化的场景和交互类型。交互式世界模型正快速发展，但现有基准仅覆盖了部分必需能力，缺乏统一的系统评估标准。为此，我们推出了 WBench，它沿视频质量、设定遵循、交互遵循、一致性和物理合规性这五个维度，对交互式世界模型进行综合评估。WBench 包含 289 个测试用例和 1,058 个交互轮次，每个用例指定一个世界设定和相应的多轮交互序列，涵盖不同场景、风格、主体，以及第一人称和第三人称视角；同时包括四种交互类型：导航、主体动作、事件编辑和视角切换。在导航方面，WBench 统一了文本控制、6-DoF 位姿控制和离散动作控制，从而支持评估具有不同原生输入接口的模型。评估采用 22 个自动计算子指标，结合了专业视觉模型和大规模多模态模型，所有指标均经过人类判断的验证。在对 20 个最先进模型的评估中，我们发现没有一个模型能在所有维度上表现卓越。我们提供了对每个模型各自优势、劣势及待解决挑战的详细诊断性洞见。代码和数据可在 https://github.com/meituan-longcat/WBench 获取。

## ProRL: Effective Reinforcement Learning for Proactive Recommendation via Rectified Policy Gradient Estimation

[ProRL: 通过修正策略梯度估计实现主动推荐的高效强化学习](https://arxiv.org/abs/2605.28293)

使用强化学习的主动推荐系统固有地面临梯度估计偏差与方差问题，本文通过逐步奖励居中化和位置特定优势估计机制加以解决。主动推荐系统 (Proactive Recommender Systems, PRSs) 旨在通过生成中间推荐路径来引导用户偏好向目标项目转移。强化学习 (Reinforcement Learning, RL) 为优化此类顺序决策任务提供了原则性框架，因为路径奖励能自然反映短期接受度与长期引导有效性。然而，直接将策略梯度应用于主动推荐系统会导致有缺陷的梯度估计。我们识别出两个缺陷：(1) 路径级奖励被分解为具有正均值的步级奖励，从而产生与路径长度相关的偏差，导致梯度偏向路径扩展而非有意义探索；(2) 对整个路径级奖励中的每一步进行加权，忽略了其分解结构，引发高梯度方差。为纠正这两个缺陷，我们提出高效强化学习框架 ProRL，包含两种主动推荐新机制。首先，逐步奖励居中化减去期望奖励以消除长度相关偏差，确保路径扩展产生零期望梯度信号。其次，位置特定优势估计利用奖励分解结构计算步相关基线，从而降低梯度方差。这些机制共同生成精确针对路径质量的策略梯度。在三个真实数据集上的实验表明，ProRL 显著优于最先进的主动推荐系统。我们的代码可在 https://github.com/hongruhou89/ProRL 获取。

## Agent Explorative Policy Optimization for Multimodal Agentic Reasoning

[面向多模态智能体推理的智能体探索性策略优化](https://arxiv.org/abs/2605.28774)

使用具备扩展推理能力的视觉语言模型 (Vision-Language Model) 的智能体 (Agent) 在工具调用方面面临挑战，AXPO 方法通过优化思考前缀 (Thinking Prefix) 和工具调用重采样 (Tool Call Resampling) 来提升性能，从而解决这些问题。这类模型虽能处理复杂问题，但许多实际问题需要借助外部工具，而仅靠内部推理往往无法应对。因此，智能体推理会交错进行两种在结构上具有不对称性的行为：思考（模型默认的独立推理行为）和工具使用（高方差辅助行为）。我们将这种不对称性称为“思考-行动差距” (Thinking-Acting Gap)。在标准的强化学习方法（如 GRPO）中，这种差距在训练过程中表现为两个诊断性症状：仅有约 30% 的轨迹 (Rollout) 中尝试了工具使用，而在尝试的组内，使用工具的轨迹中约有 40% 的问题全部出错，这削弱了需要工具调用处的学习信号。为此，我们提出 AXPO（Agent eXplorative Policy Optimization）：对于每个全部出错的工具使用子组，AXPO 固定思考前缀，重新采样工具调用及其后续内容，并结合基于不确定性的前缀选择方法。在九个多模态基准测试和三个规模版本的 Qwen3-VL-Thinking 上，SFT+AXPO 在平均性能上优于 SFT+GRPO（8B 模型上平均 Pass@1 提升 1.8 个百分点，Pass@4 提升 1.8 个百分点），且 8B 规模的 SFT+AXPO 在 Pass@4 上以少 4 倍的参数量超越了 32B 基础模型 (Base Model)。

## Macaron-A2UI: A Model for Generative UI in Personal Agents

[Macaron-A2UI: 个人智能体中的生成式 UI 模型](https://arxiv.org/abs/2605.24830)

生成式 UI 模型使个人智能体 (Personal Agent) 能够生成带有轻量级可执行动作的动态界面，从而超越纯文本交互，实现更丰富的用户体验。随着个人智能体在处理复杂、以用户为中心的任务上不断演进，静态的纯文本对话迅速成为瓶颈。生成式 UI (Generative UI) 作为一种新的必要接口层，能够根据交互上下文实时动态地合成合适的控件、选项和状态。我们提出了 Macaron-A2UI，一种用于个人智能体的生成式 UI 模型。其目标是超越纯文本交互，使智能体既能生成自然语言，也能生成轻量级可执行的 UI 动作，以支持信息采集、偏好精炼、确认以及多目标组织。我们从异构对话来源构建了一个大规模生成式 UI 语料库，引入了用于受控评估的 A2UI-Bench 基准，并通过基于 LoRA 的参数高效有监督微调 (Supervised Fine-tuning) 结合基于奖励的强化学习 (Reward-driven Reinforcement Learning)，训练了 30B、235B 和 754B 参数的模型。最佳的 Macaron-A2UI 模型在无显式模式提示 (schema hints) 的情况下，在 A2UI-Bench 上取得了 75.6 的总体分数，超越了最强的全模式 (Full-schema) 前沿基线模型。我们开源了模型、基准测试和评估协议，以支持未来个人智能体中生成式 UI 的相关研究。

自主智能体正在从工具演变为社会基础设施层：它们浏览网页、采购商品、部署软件、管理系统，并越来越多地相互交互。随着这些系统的规模化，瓶颈已从原始模型能力转向协调问题。智能体需要建立可靠的关系、组织多智能体工作、交换价值、支持AI经济，并在现实世界的监管下保持安全和可问责。本文介绍了Foundation Protocol (FP)，一个面向新兴人机社会的图优先协调层。FP统一了异构实体，包括智能体、工具、资源、人类、机构和组织，并原生支持多方组织和基于事件的协作。它还提供了用于计量、收据和结算的经济原语，并将策略、溯源和审计视为一级关注点。FP的设计目标是包装和桥接现有协议而非取代它们，从而在减少集成和治理开销的同时实现渐进式采用。其目的是在保持自主智能体可组合性的同时，让可问责性成为不可妥协的底线，从而使协调本身能够成为开放、多元且可治理的人机社会的共享基础设施。

## From Pixels to Words -- Towards Native One-Vision Models at Scale

NEO-ov 是一个原生视觉语言模型 (native vision-language model)，它以端到端方式学习跨帧与像素-词对应关系，无需任何模块化组件，从而支持统一的时空建模，并在视觉感知任务中达到有竞争力的性能。当前的视觉语言模型 (VLMs) 通常将独立的图像编码器与语言解码器通过多阶段对齐拼接在一起，这种模块化架构不可避免地导致像素级信号在帧间碎片化，并延迟了早期的像素-词交互。与此同时，尽管原生视觉语言模型在单图像任务上表现出色，但在多图像、视频理解和空间智能方面仍鲜有探索。为此，我们提出 NEO-ov —— 一个端到端学习跨帧与像素-词对应关系的原生基础模型，无需任何外部编码器、辅助适配器或事后融合。通过彻底消除模块边界，NEO-ov 使细粒度和统一的时空建模能力自然地内生于模型内部。值得注意的是，NEO-ov 在细粒度视觉感知上表现突出，同时显著缩小了与模块化模型的性能差距，验证了原生"一视"架构不仅在规模化上可行，而且具备竞争力。除实证性能外，我们还提供了系统的架构分析与详细的训练指南，以推动后续原生多模态建模研究。我们的代码与模型已开源在：https://github.com/EvolvingLMMs-Lab/NEO。

## SpatialBench: Is Your Spatial Foundation Model an All-Round Player?

[SpatialBench: 你的空间基础模型是全能型选手吗？](https://arxiv.org/abs/2605.27367)

SpatialBench 提出了一个全面的基准，用于在不同领域和任务中评估空间基础模型，揭示了当前模型的局限性，并引入了 DA-Next-5M 和 DA-Next 以推进空间表征学习。尽管空间基础模型在标准数据集上展现了令人印象深刻的性能，但一个关键问题仍然存在：它们是否真正是全能型选手，能够稳健泛化到多样化的下游任务、任意视角、变化的场景领域、不同的输入密度以及特定的硬件约束？要回答这个总体问题，需要一个整体评估，然而当前模型主要在其专门设计或训练的特定领域上进行评估。这种评估本质上受到狭窄的范式覆盖、有限的场景领域和任意帧采样的限制，使得从根本上难以评估其真实泛化能力。为了解决这一空白，我们提出了 SpatialBench，一个用于空间基础模型的跨范式、多领域基准，采用确定性采样。SpatialBench 具有前所未有的规模和严格的确定性设计，包含 19 个数据集和 546 个场景，覆盖 5 个不同的空间领域。它在 4 种不同的输入密度设置下，对 6 种范式中的 41 个模型在 5 个任务套件上进行了全面评估。我们的广泛评估揭示，当前模型尚未成为全能型选手，并为未来发展指明了关键方向。具体来说，我们证明了完全上下文注意力最大化准确性，而有限内存策略解锁了长序列的可扩展性。此外，我们在具有挑战性的具身化和自我中心任务中的实证评估表明，严格的领域对齐和高数据质量对性能的影响远大于简单的数据集扩展。最后，为了解决我们分析中识别的最大数据缺口，我们超越了评估，引入了一个大规模数据集 DA-Next-5M 和一个强基线模型 DA-Next，从而拓展了空间表征学习的边界。

## Self-Improving Language Models with Bidirectional Evolutionary Search

[自我改进语言模型与双向进化搜索](https://arxiv.org/abs/2605.28814)

双向进化搜索通过融合前向候选进化与后向目标分解，克服传统搜索方法的局限性，从而提升语言模型的生成能力。搜索已被视为一种有效的自我改进方法，适用于语言模型和智能体系统，既可用于后训练样本生成，也可用于推理阶段。然而，广泛采用的"N中选优"采样和树搜索等方法存在两个根本性缺陷：它们依赖稀疏的验证信号，且主要通过自回归扩展构建候选，这限制了探索范围，使其仅能停留在模型概率质量较高的区域。为解决这些问题，我们提出双向进化搜索（BES, Bidirectional Evolutionary Search），该框架将前向候选进化与后向目标分解相结合。在前向搜索中，BES通过进化算子增强标准扩展：这些算子将部分轨迹重新组合，生成难以从单次模型展开中获得的候选解。在后向搜索中，BES递归地将原始任务分解为可验证的子目标，提供密集的中间反馈以指导前向搜索。我们提供理论依据表明：仅依赖扩展搜索生成的候选被限制在狭窄的熵壳（entropy shell）内，而进化算子能够突破这一限制；同时，后向搜索能以指数级减少找到正确答案所需的样本数量。实验结果显示，在主流后训练算法无法带来改进的挑战性任务上，BES能持续取得性能提升；在推理阶段的三个开放问题求解基准测试中，BES在平均性能和最佳性能两方面均优于现有的开源框架。代码和训练模型已在 https://github.com/Embodied-Minds-Lab/BES 开源。

## CollectionLoRA: Collecting 50 Effects in 1 LoRA via Multi-Teacher On-Policy Distillation

[CollectionLoRA: 通过多教师在线策略蒸馏将50种效果收集到一个LoRA中](https://arxiv.org/abs/2605.25378)

CollectionLoRA通过多教师蒸馏与专门的概念隔离和生成机制，将多个LoRA的知识压缩到单个模型中，实现了多种定制图像编辑效果的高效部署。定制图像编辑旨在利用有限的配对数据，通过低秩适配 (Low-Rank Adaptation, LoRA) 技术，为预训练扩散模型赋予特定的视觉效果。随着所需效果数量的增长，存储和动态加载众多效果LoRA会显著增加部署成本。此外，当前流程通常将这些效果LoRA与加速模块级联以实现快速生成，这引发严重的参数干扰，导致概念混淆 (concept bleeding) 和风格退化。我们提出CollectionLoRA，这是一个多教师在线策略蒸馏框架，能够将多达50种不同效果LoRA的概念以及少步生成能力蒸馏到一个LoRA中。这从根本上解决了特征干扰问题，并大幅降低了部署成本。具体而言，该方法引入了：(i) 概率双流路由 (Probabilistic Dual-Stream Routing) 机制，使模型在训练过程中能随机切换数据源，有效增强其在未见场景中的泛化能力；(ii) 非对称正交提示 (Asymmetric Orthogonal Prompting) 策略，在提示空间内实现概念隔离；(iii) 从粗到精的蒸馏目标 (Coarse-to-Fine Distillation Objective)，以缩小教师与学生模型之间的分布差异。大量实验表明，CollectionLoRA将所有定制效果和少步生成能力蒸馏到单个LoRA中，在降低部署开销的同时，实现了与独立训练的教师模型相当或更优的概念保真度。

## TriSplat: Simulation-Ready Feed-Forward 3D Scene Reconstruction

[TriSplat: 可直接用于仿真的前馈式 3D 场景重建](https://arxiv.org/abs/2605.26115)

TriSplat 是一种前馈式 3D 重建网络，它利用有向三角形基元，从单张图像直接生成可用于仿真的网格，从而省去了昂贵的后处理环节。目前，稀疏视角的 3D 重建越来越多地采用前馈式喷射 (splatting) 网络，这类网络直接从图像预测显式基元。然而，现有方法大多仍基于高斯基元，且只能间接表达表面：若想提取可供下游仿真、物理推理或具身交互使用的有效网格，仍需进行繁琐的后处理步骤，这违背了前馈式方法的初衷。该问题在无姿态设定下尤为突出，因为场景结构和相机参数必须从稀疏观测中联合估计。为此，我们提出 TriSplat，这是一种前馈式重建网络，它使用有向三角形基元表示场景，并仅通过一次前向传播即可直接输出可用于仿真的网格场景。给定输入图像，网络会预测局部 3D 点图、三角形属性、相机姿态以及可选的内参。与将三角形方向作为无约束潜变量进行回归不同，我们的方法从预测的点图中提取几何法线，借助图像条件法线网络头 (normal head) 对其进行优化，再将其转换为稳定的局部坐标系，用于三角形参数化。单法线引导 (mono-normal bootstrap) 策略能稳定早期训练过程，而不透明度与模糊度调度方案则逐步锐化学习到的表面表示，便于直接提取网格。在 RealEstate10K 和 DL3DV 上的实验表明，与基于高斯的前馈基线相比，这种表示方法能生成几何结构更准确的重建结果，同时保持有竞争力的新视角渲染质量。由于渲染基元本身就是表面三角形，输出结果可直接被物理引擎、碰撞检测器以及标准渲染管线使用，无需任何转换，因而它成为前馈式 3D 场景重建中一种实用的即用型仿真解决方案。

## minWM: A Full-Stack Open-Source Framework for Real-Time Interactive Video World Models

[minWM: 全栈开源实时交互式视频世界模型框架](https://arxiv.org/abs/2605.30263)

本文提出一种通用框架，通过微调与蒸馏技术，将双向视频扩散模型转换为具备可控性、因果性及低延迟特性的实时交互式世界模型。近年来，视频扩散基础模型在高质量视频生成方面取得了显著突破，但将其转化为实时交互式视频世界模型仍面临巨大挑战。交互式世界模型需满足可控、因果和低延迟的推理生成要求，这在实践中需要构建涵盖数据构建、可控微调、自回归训练、少步蒸馏和流式推理的完整流程。为此，我们提出 minWM——一个用于构建实时交互式视频世界模型的全栈开源框架。minWM 提供端到端流程，将现有的双向 T2V/TI2V 视频基础模型转换为相机可控的少步自回归世界模型。具体而言，minWM 首先对带相机控制的双向视频扩散模型进行微调，然后应用 Causal Forcing / Causal Forcing++ 流程（包括自回归（AR）扩散训练、因果 ODE 或因果一致性蒸馏以及非对称 DMD）将其蒸馏为少步自回归生成器，以实现低延迟推理。该框架采用模块化且架构可扩展的设计：我们在两个代表性开源主干网络上实现了该框架，包括 Wan2.1-T2V-1.3B 和 HY1.5-TI2V-8B，分别覆盖了基于交叉注意力的条件注入和 MMDiT 风格架构。此外，minWM 支持将现有视频世界模型（如 HY-WorldPlay）适配到新的数据分布、训练方案和延迟目标。除了提供运行脚本、模型检查点、文档和推理代码外，我们还针对相机轨迹质量、可控性训练步骤及最小批大小要求进行了实用消融实验。我们希望 minWM 能成为构建和适配实时交互式视频世界模型的一个可复现、可扩展的方案。项目页面：[https://github.com/shengshu-ai/minWM](https://github.com/shengshu-ai/minWM)

## StepAudio 2.5 Technical Report

[StepAudio 2.5 技术报告](https://arxiv.org/abs/2605.23463)

StepAudio 2.5 是一种统一的音频-语言模型，通过采用针对特定任务的人类反馈强化学习 (Task-tailored Reinforcement Learning from Human Feedback, RLHF) 来优化不同操作模式下的共享表示，从而在自动语音识别 (ASR)、文本转语音合成 (TTS) 和实时口语交互方面达到专用系统的水平。统一的音频-语言建模已成为现代语音系统的主流趋势，旨在将大语言模型的推理能力引入听觉任务。然而，现有的统一基础模型往往难以在 ASR、TTS 和实时交互等各项能力上匹敌专用系统的深度。弥合这一差距仍是一项开放挑战。本报告介绍了 StepAudio 2.5——一个统一的音频-语言基础模型，其在上述三类能力上均能匹配或超越专用系统。我们并未将这些任务视为架构上独立的问题，而是基于以下前提：一旦文本和音频共享一个多模态表示空间，任务专业化便归结为操作模式上的差异——即数据构建、优化目标和解码约束。基于这一洞察，我们将后训练范式从标准监督学习升级为针对特定任务的 RLHF，并将其作为定义复杂优化目标的核心机制。借助以 RLHF 为中心的对齐策略以及专门的解码方法，我们将共享的骨干模型塑造成三种不同的操作模式。具体而言，ASR 分支通过可验证的多 Token 解码 (Verifiable Multi-Token Decoding) 提升转录效率；TTS 分支利用基于偏好的 RLHF 和上下文丰富的监督信息，实现可控且富有表现力的合成；实时分支则在 RLHF 框架内采用生成式奖励建模 (Generative Reward Modeling)，达成低延迟且角色一致的对话。在标准基准测试中，StepAudio 2.5 在 ASR、TTS 和实时交互任务上均取得了最先进的结果，证明一个单一的音频-语言基础模型能够成功整合语音理解、生成与实时交互的不同部署目标。

## ResearchMath-14K: Scaling Research-Level Mathematics via Agents

[ResearchMath-14K: 通过智能体扩展研究级数学](https://arxiv.org/abs/2605.28003)

为了推动语言模型在研究级数学推理方面的发展，我们引入了 ResearchMath-14k 数据集和 ResearchMath-Reasoning 推理路径。研究结果表明，经过筛选的开放问题尝试能够为模型改进提供有效的监督信号。数学的最前沿由那些解法尚未知晓的问题所界定，但语言模型能否在没有人类干预的情况下有意义地处理这类问题，目前仍不明确。一个主要障碍是缺乏大规模的研究级数学数据集。为此，我们提出了 ResearchMath-14k，这是一个包含 14,056 个问题的集合，通过多智能体流水线从学术来源中精心筛选而成，是迄今为止规模最大的研究级数学问题集。此外，我们还生成了 ResearchMath-Reasoning，即从两个开放模型中提取的 22 万条教师推理轨迹。在这些轨迹中，我们观察到常见的回避行为，例如放弃尝试和虚构引用。有趣的是，在八个开放权重模型中，较新版本每条轨迹产生的参考文献数量是旧版本的 5.6 倍，虚假引用数量是旧版本的 5.0 倍。对 ResearchMath-Reasoning 进行基于智能体的过滤之后，我们将 Qwen3 模型从 4B 参数微调到 30B 参数，使其性能平均比基础模型提升了 9.2 个百分点。这证明了，即使在完全缺少正确推理轨迹的情况下，经过筛选的开放问题尝试仍能提供有价值的监督信息。我们将公开提供 ResearchMath-14k 数据集，以支持未来在研究级数学推理领域的工作。
