## MinT: Managed Infrastructure for Training and Serving Millions of LLMs

[MinT：面向百万级 LLM 训练与服务的托管基础设施](https://arxiv.org/abs/2605.13779)

MinT（MindLab Toolkit）是一个面向 LoRA 后训练和在线推理的托管基础设施系统。它针对少量大型基础模型部署下产生大量训练策略的场景，通过保持基础模型常驻、仅移动轻量级 LoRA adapter 来完成 rollout、更新、导出、评估、服务和回滚，将分布式训练、服务调度和数据移动隐藏在服务接口之后。MinT 在三个维度进行扩展：Scale Up 将 LoRA RL 扩展至前沿规模的稠密和 MoE 架构（包括 MLA 和 DSA 注意力路径），训练和服务验证超过 1T 总参数；Scale Down 仅移动导出的 LoRA adapter（在 rank-1 设置中不到基础模型的 1%），adapter-only 传输使 4B 稠密模型上的步进速度提升 18.3 倍、30B MoE 上提升 2.85 倍，同时并发多策略 GRPO 将实际耗时缩短 1.77 倍和 1.45 倍而不增加峰值内存；Scale Out 将持久策略寻址与 CPU/GPU 工作集分离，单引擎支持 10^6 规模的策略编目（实测扫描 100K），集群规模支持千 adapter 并行激活，冷加载作为调度服务工作处理，打包 MoE LoRA 张量将在线引擎加载效率提升 8.5-8.7 倍。

## Mean Mode Screaming: Mean--Variance Split Residuals for 1000-Layer Diffusion Transformers

[Mean Mode Screaming：面向 1000 层扩散 Transformer 的均值-方差分裂残差](https://arxiv.org/abs/2605.06169)

将扩散 Transformer（DiT）扩展到数百层会引入一种结构性脆弱：网络可能进入一种隐性的均值主导崩塌状态，使 token 表征同质化并抑制中心化变化。通过机制审计，作者将这种崩塌的触发事件定位为 Mean Mode Screaming（MMS）。MMS 甚至在训练看似稳定时也会发生——残差写入器上的均值相干反向冲击打开深层残差分支，驱动网络进入均值主导状态。这一行为源于梯度分解为均值相干和中心化两个分量，加之值同质化后 Softmax Jacobian 零空间对注意力逻辑梯度的结构性抑制。为此，作者提出 Mean-Variance Split（MV-Split）残差，将单独增益的中心化残差更新与漏斗式主干均值替换相结合。在 400 层单流 DiT 上，MV-Split 阻止了导致未稳定基线崩溃的散逸性崩塌，同时在整个训练计划中显著优于 LayerScale 等 token 各向同性门控方法。最终，作者展示了 1000 层 DiT 作为边界尺度的验证运行，证明架构在极端深度下仍可稳定训练。

## MemLens: Benchmarking Multimodal Long-Term Memory in Large Vision-Language Models

[MemLens：大型视觉语言模型多模态长期记忆基准评测](https://arxiv.org/abs/2605.14906)

MemLens 是一个面向多模态多轮对话记忆的综合基准，包含 789 个问题，覆盖五种记忆能力（信息提取、多轮推理、时间推理、知识更新和答案拒绝），在四种标准上下文长度（32K-256K token）下评估。图像消融研究证实 80.4% 的问题需要视觉证据。对 27 个 LVLM 和 7 个记忆增强 Agent 的评估发现：长上下文 LVLM 在短上下文中通过直接视觉锚定取得高准确率，但随着对话增长而退化；记忆 Agent 在长度上稳定但在存储压缩下损失视觉保真度。多轮推理使大多数系统低于 30%，单独一种方法都无法解决任务，激发了对结合长上下文注意力和结构化多模态检索的混合架构的需求。

## LLMs Improving LLMs: Agentic Discovery for Test-Time Scaling

[LLM 改善 LLM：面向测试时扩展的 Agent 自动发现](https://arxiv.org/abs/2605.08083)

AutoTTS 改变了研究者的设计对象：从手工设计 TTS 启发式策略到构建 TTS 策略可自动发现的环境。核心在于将宽度-深度 TTS 表述为对预收集推理轨迹和探测信号的控制器合成，控制器决定何时分支、继续、探测、剪枝或停止，且无需重复 LLM 调用即可廉价评估。作者进一步引入 beta 参数化使搜索可处理，并使用细粒度执行追踪反馈提高发现效率。在数学推理基准上，发现的策略改善了整体准确率-成本权衡，泛化至未见基准和模型规模，而整个发现过程仅需 $39.9 和 160 分钟。

## HyperEyes: Dual-Grained Efficiency-Aware Reinforcement Learning for Parallel Multimodal Search Agents

[HyperEyes：面向并行多模态搜索 Agent 的双粒度效率感知强化学习](https://arxiv.org/abs/2605.07177)

HyperEyes 将视觉锚定和检索融合为单一原子动作，实现在一轮内对多实体并发搜索，同时将推理效率视为一等训练目标。冷启动阶段通过并行数据合成流水线提供监督，核心贡献双粒度效率感知 RL 框架在宏观层面提出 TRACE（工具使用参考自适应成本效率）轨迹级奖励，在微观层面适配在线策略蒸馏注入密集 token 级修正信号。作者还提出 IMEB 人工策划基准联合评估搜索能力和效率。在六个基准上，HyperEyes-30B 以平均少 5.3 倍工具调用轮次超越最强可比开源 Agent 9.9% 准确率。

## EVA-Bench: A New End-to-end Framework for Evaluating Voice Agents

[EVA-Bench：面向语音 Agent 的端到端评估框架](https://arxiv.org/abs/2605.13841)

EVA-Bench 是一个端到端评估框架，在模拟侧编排 Bot-Bot 音频对话并自动检测用户模拟器错误，在测量侧引入两个复合指标：EVA-A（准确率，涵盖任务完成、忠实度和音频级语音保真度）和 EVA-X（体验，涵盖对话进展、口语简洁性和轮替时序）。基准包含 213 个跨三个企业域的场景、口音和噪声鲁棒性受控扰动套件，以及区分峰值与可靠能力的 pass@1/pass@k/pass^k 测量。12 个系统评估发现：没有系统同时在 EVA-A 和 EVA-X pass@1 上超过 0.5；峰值与可靠性能差距显著（EVA-A 上中位数 0.44）；口音和噪声扰动暴露出显著鲁棒性差距。

## World Action Models: The Next Frontier in Embodied AI

[世界动作模型：具身 AI 的下一个前沿](https://arxiv.org/abs/2605.12090)

视觉-语言-动作（VLA）模型在具身策略学习中取得了强语义泛化，但学习的是从观测到动作的反应式映射，未显式建模物理世界在干预下的演化。世界动作模型（WAMs）统一了预测性状态建模和动作生成，面向未来状态和动作的联合分布而非仅动作。本文形式化定义 WAMs，追溯 VLA 和世界模型研究的基础和早期整合，将现有方法组织为级联和联合 WAMs 的结构化分类法（按生成模态、条件机制和动作解码策略进一步细分），系统分析从机器人遥操作、便携人类演示、仿真到互联网规模自我中心视频的数据生态系统，并综合视觉保真度、物理常识和动作合理性的评估协议。

## Do Enterprise Systems Need Learned World Models? The Importance of Context to Infer Dynamics

[企业系统需要学习型世界模型吗？上下文对推断动态的重要性](https://arxiv.org/abs/2605.12178)

世界模型使 Agent 通过内化环境动态来预测动作效果，但在企业系统中，动态往往由租户特定的业务逻辑定义，随部署而异且持续演变。作者提出企业发现 Agent——在运行时通过读取系统配置恢复相关的转移动态，而非仅依赖内化表征。在 CascadeBench 基准和部署偏移评估中，离线训练的世界模型在分布内表现良好，但随着动态变化而退化；基于发现的 Agent 通过将预测锚定在当前实例上更具鲁棒性。结果表明在可配置企业环境中，Agent 不应仅依赖固定的内化动态。

## SANA-WM: Efficient Minute-Scale World Modeling with Hybrid Linear Diffusion Transformer

[SANA-WM：基于混合线性扩散 Transformer 的高效分钟级世界建模](https://arxiv.org/abs/2605.15178)

SANA-WM 是一个高效的 2.6B 参数开源世界模型，原生训练用于一分钟生成，合成高保真 720p 分钟级视频并支持精确相机控制。四项核心设计驱动架构：（1）混合线性注意力结合逐帧 Gated DeltaNet 和 softmax 注意力实现内存高效的长上下文建模；（2）双分支相机控制确保精确 6-DoF 轨迹遵循；（3）两阶段生成流水线通过长视频精炼器提升质量和连贯性；（4）鲁棒标注流水线从公开视频提取精确公制 6-DoF 相机姿态。SANA-WM 仅使用约 213K 公开视频片段，64 张 H100 上 15 天完成训练，单 GPU 生成 60 秒片段；蒸馏变体在单张 RTX 5090 上 34 秒完成去噪。在分钟级世界模型基准上，SANA-WM 在动作遵循准确率上超越先前开源基线，视觉质量与工业基线可比，吞吐量提升 36 倍。

## Darwin Family: MRI-Trust-Weighted Evolutionary Merging for Training-Free Scaling of Language-Model Reasoning

[Darwin Family：面向无训练语言模型推理扩展的 MRI 信任加权进化合并](https://arxiv.org/abs/2605.14386)

Darwin Family 通过无梯度的权重空间重组实现 LLM 的无训练进化合并。Darwin 引入三个关键思想：（1）14 维自适应合并基因组实现细粒度组件和块级重组；（2）MRI-Trust Fusion 通过可学习信任参数自适应平衡诊断层重要性信号与进化搜索；（3）架构映射器实现异构模型家族间的跨架构繁殖。旗舰 Darwin-27B-Opus 在 GPQA Diamond 上达 86.9%，在 1252 个评测模型中排名第 6，超越其完全训练的基础模型而无需任何基于梯度的训练。从 4B 到 35B 参数规模，Darwin 模型一致超越其父模型，支持递归多代进化和 Transformer-Mamba 组件的无训练合并。

## HumanNet: Scaling Human-centric Video Learning to One Million Hours

[HumanNet：将以人为中心的视频学习扩展到一百万小时](https://arxiv.org/abs/2605.06747)

HumanNet 是一个百万小时级别的以人为中心的视频语料库，捕捉人类如何与物理世界大规模交互。数据集涵盖第一人称和第三人称视角，覆盖细粒度活动、人-物交互、工具使用和跨多样真实环境的长程行为，并提供以交互为中心的标注（描述、运动描述和手部/身体相关信号）。在固定验证数据集上的 VLA 消融实验中，用 HumanNet 中 1000 小时自我中心视频继续训练 Qwen VLM 超过用 100 小时 Magic Cobot 真实机器人数据继续训练的效果，表明自我中心人类视频可能是机器人数据的可扩展且经济高效的替代方案。

## Qwen-Image-VAE-2.0 Technical Report

[Qwen-Image-VAE-2.0 技术报告](https://arxiv.org/abs/2605.13565)

Qwen-Image-VAE-2.0 是一套高压缩变分自编码器（VAE），在重建保真度和可扩散性方面取得显著进展。为解决高压缩下的重建瓶颈，采用包含全局跳跃连接（GSC）和扩展潜在通道的改进架构；将训练扩展至数十亿图像并引入合成渲染引擎提升文字密集场景表现；实现增强语义对齐策略使潜在空间高度适配扩散建模；利用非对称无注意力编码器-解码器骨干最小化编码开销。作者还提出 OmniDoc-TokenBench 基准用于评估文字密集场景性能。Qwen-Image-VAE-2.0 在高压缩比下达到 SOTA 重建性能，下游 DiT 实验显示其具有卓越的可扩散性，显著加速收敛。

## Geometry Conflict: Explaining and Controlling Forgetting in LLM Continual Post-Training

[几何冲突：解释和控制 LLM 持续后训练中的遗忘](https://arxiv.org/abs/2605.09608)

本文通过任务几何研究 LLM 持续后训练：将每个后训练任务表示为其参数更新，研究更新诱导的协方差几何。核心发现是：遗忘可被视为状态相对的更新整合失败，当任务诱导的协方差几何与演化模型状态的几何不对齐时就会发生。顺序更新在与先前更新塑造的模型状态保持兼容时发生迁移，在状态相对几何冲突变高时产生干扰。基于此发现，作者提出几何冲突 Wasserstein 合并（GCWM）——一种无数据更新整合方法，通过高斯 Wasserstein 重心构造共享 Wasserstein 度量，并用几何冲突门控几何感知校正。在 Qwen3 0.6B-14B 上，GCWM 一致超越无数据基线。

## MemEye: A Visual-Centric Evaluation Framework for Multimodal Agent Memory

[MemEye：以视觉为中心的多模态 Agent 记忆评估框架](https://arxiv.org/abs/2605.15128)

MemEye 从两个维度评估记忆能力：一是衡量决定性视觉证据的粒度（从场景级到像素级），二是衡量检索证据的使用方式（从单一证据到进化性综合）。在该框架下构建了覆盖 8 个生活场景任务的新基准，配有消融驱动的验证门评估可回答性、捷径抗性、视觉必要性和推理结构。对 13 种记忆方法和 4 种 VLM 骨干的评估表明，当前架构在保留细粒度视觉细节和推理状态变化方面仍有困难，长期多模态记忆依赖于证据路由、时间跟踪和细节提取。

## TMAS: Scaling Test-Time Compute via Multi-Agent Synergy

[TMAS：通过多 Agent 协同扩展测试时计算](https://arxiv.org/abs/2605.10344)

TMAS 将推理组织为专业化 Agent 间的协作过程，实现跨 Agent、轨迹和精炼迭代的结构化信息流。为支持有效的跨轨迹协作，TMAS 引入分层记忆：经验库复用低层可靠中间结论和局部反馈，准则库记录先前探索的高层策略以引导后续搜索远离冗余推理模式。此外，设计了混合奖励 RL 方案，联合保持基础推理能力、增强经验利用并鼓励探索。在多个挑战性推理基准上，TMAS 实现比现有测试时扩展基线更强的迭代扩展，混合奖励训练进一步提升迭代间的扩展有效性和稳定性。
